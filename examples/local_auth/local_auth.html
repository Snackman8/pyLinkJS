Local Authentication Example

A professional baseline for local username/password access backed by SQLite, including a one-time bootstrap flow and administrator user management.
Admin Users Logout
Overview

This example uses pluginLocalAuth with a SQLite database stored next to the application. On first run, visit /admin/users to complete the one-time bootstrap flow and create the first administrator account.

Bootstrap
If the database is empty, the admin route opens a one-time first-admin setup page.
Administration
Administrators can create or delete users through the built-in management console.
